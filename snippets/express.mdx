---
name: Express.js
order: 1
---

Install with yarn:

```sh
yarn add @apilytics/express
```

Install with npm:

```sh
npm install @apilytics/express
```

Usage:

```javascript
// server.js

const { apilyticsMiddleware } = require('@apilytics/express');
const express = require('express');

const app = express();

app.use(apilyticsMiddleware(process.env.APILYTICS_API_KEY));
```
