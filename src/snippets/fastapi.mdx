Install with pip:

```sh
pip install apilytics
```

Install with poetry:

```sh
poetry add apilytics
```

Usage:

```python
# main.py

import os

from apilytics.fastapi import ApilyticsMiddleware
from fastapi import FastAPI

app = FastAPI()

app.add_middleware(ApilyticsMiddleware, api_key=os.getenv("APILYTICS_API_KEY"))
```
